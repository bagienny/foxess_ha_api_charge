# foxess_ha_api_charge
Simple HA component for modifying FoxESS Charge Times using OpenAPI

Feel free to fork

# Installation using HACS
1. add custom repository https://github.com/bagienny/foxess_ha_api_charge
2. download FoxESS API Charge Periods
3. restart Home Assistant
4. go to Settings -> Devices and Services -> Add integration -> FoxESS API Charge Periods
5. provide API key and device SN
